---
description: Review current XAU/USD runtime status and recent trade context
---
Use the operator CLI to review current context:

1. Status
```
!`node cli.js status`
```

2. Open trades
```
!`node cli.js open-trades`
```

3. Recent trades
```
!`node cli.js recent-trades --limit 5`
```

Summarise:
- whether the runtime is currently active or constrained by cooldown/risk
- what open trades are already in flight
- whether recent outcomes suggest caution or confidence
